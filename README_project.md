# 💰 Finance Project

Welcome to the repository for the finance application. This project is built using modern web technologies featuring Next.js, React, Tailwind CSS, and TypeScript.

## 💻 Run the Project Locally

### 1. Prerequisites

Before starting, ensure you have [Node.js](https://nodejs.org/) installed on your computer.

### 2. Clone the Repository

```bash
git clone https://github.com/Amsapraba/finance-project.git
```

### 3. Navigate to the Project Folder

```bash
cd finance-project/creditops-studio
```

### 4. Install Dependencies

```bash
npm install
```

### 5. Start the Development Server

```bash
npm run dev
```

### 6. View the Application

Open your browser and visit:

```
http://localhost:3000
```

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Library:** React
- **Language:** TypeScript
- **Styling:** Tailwind CSS
